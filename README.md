# lux-akademie.de

Official website of **LUX – akademie neues bauhaus brandenburg e.V.**
Live at <https://lux-akademie.de>.

## Stack

Plain HTML / CSS / JS — no build step, no framework.
Bilingual DE/EN with a language toggle persisted in `localStorage`.
Hosted on GitHub Pages from `main` (root).

## Local preview

```sh
python3 -m http.server 8080
```

Then open <http://localhost:8080/>.

## Structure

```
index.html, programme.html, ort.html, meister.html, myzel.html,
kontakt.html, newsletter.html, impressum.html, datenschutz.html
robots.txt, sitemap.xml
assets/
  css/main.css
  js/main.js
  img/
```
